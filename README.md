## Installation

Follow these steps to set up the project:

```bash
git clone https://github.com/neonsodium/instant-ai.git

cd instant-ai

python3 -m venv venv

source venv/bin/activate

pip3 install -r requirements.txt
```

## Usage

To run the project, execute the following command:

```bash
python3 run.py
```
